# ⚡ Focus Flow - Quick Start

Get started with Focus Flow in 2 minutes!

## 🚀 Option 1: Install from URL (Easiest)

1. **Open AI Edge Gallery** on your Android/iOS device
2. Go to **Agent Skills** tab
3. Tap the **(+)** button
4. Select **"Load skill from URL"**
5. Enter: `https://podzemniytip.github.io/focus-flow/`
   _(Replace podzemniytip with actual GitHub username)_
6. **Done!** The skill is ready to use

## 📱 Option 2: Local Install (No Internet)

### On Windows:
```powershell
# Connect your Android device via USB
adb push focus-flow/ /sdcard/Download/
```

### In the App:
1. Open AI Edge Gallery
2. Go to Agent Skills
3. Tap **(+)** → **"Import local skill"**
4. Navigate to **Download/focus-flow**
5. Select the folder
6. **Done!**

## 🎯 How to Use

### Voice Commands:

Just talk to Gemma naturally:

**Starting focus sessions:**
- "Start a 25 minute focus session"
- "I need to focus for an hour"
- "Help me concentrate on writing"

**Taking breaks:**
- "I need a break"
- "Start a 5 minute break timer"

**Tracking habits:**
- "Show my daily habits"
- "Track my habits"

**Managing goals:**
- "What are my goals today?"
- "Show my tasks"

**Viewing stats:**
- "Show my productivity statistics"
- "How much have I focused this week?"

### Manual Controls:

In the UI, you can:
- **Timer tab**: Choose duration (15/25/45/60 min), start/pause/reset
- **Habits tab**: Add habits, check them off daily, build streaks
- **Goals tab**: Add daily goals, mark as complete
- **Stats tab**: View all your productivity data
- **Dark mode**: Toggle with 🌙 button (or auto-detect system theme)

## 🎨 Tips

1. **Dark Mode**: Automatically matches your system, or toggle manually
2. **Streaks**: Check off habits daily to build 🔥 fire streaks
3. **Custom Durations**: Use 15/25/45/60 min buttons or ask for custom
4. **Audio**: First time, click start to enable sound (browser security)
5. **Data**: Never deleted unless you clear app storage

## 🐛 Troubleshooting

**No sound?**
- Click start once to enable audio (browser requirement)

**Data lost?**
- Check device storage wasn't cleared
- Data is device-specific (not synced)

**Dark mode stuck?**
- Refresh page or toggle manually with 🌙

**Timer won't start?**
- Refresh the page
- Check webview permissions
