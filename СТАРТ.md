## 📞 Если что-то не работает

1. **Проблемы с git:**
   - Убедитесь что git установлен: `git --version`
   - Создайте Personal Access Token на GitHub если нужен

2. **GitHub Pages не работает:**
   - Проверьте что `.nojekyll` файл есть
   - Подождите 5-10 минут после включения
   - Проверьте что репо public

3. **Модалка шеринга не появляется:**
   - Проверьте console в браузере (F12)
   - Убедитесь что это ПЕРВАЯ завершенная сессия
   - Очистите localStorage: `localStorage.clear()`

4. **Twitter не открывается:**
   - Проверьте что popup не заблокирован браузером
   - Попробуйте в другом браузере
